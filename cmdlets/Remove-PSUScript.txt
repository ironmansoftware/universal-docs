
NAME
    Remove-PSUScript
    
SYNOPSIS
    Removes a script.
    
    
SYNTAX
    Remove-PSUScript [-Id] <Int64> [-AppToken <String>] [-ComputerName <String>] [-Integrated] [-UseDefaultCredentials] [<CommonParameters>]
    
    Remove-PSUScript [-AppToken <String>] [-ComputerName <String>] [-Integrated] -Script <Script> [-UseDefaultCredentials] [<CommonParameters>]
    
    
DESCRIPTION
    Removes a script. This also deletes the script from the PSU internal git repo as well as an optionally configured remote.
