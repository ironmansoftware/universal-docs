
NAME
    Get-UAJobFeedback
    
SYNOPSIS
    Returns feedback requested by jobs.
    
    
SYNTAX
    Get-UAJobFeedback [-AppToken <String>] [-ComputerName <String>] -Job <Job> [-UseDefaultCredentials] [<CommonParameters>]
    
    Get-UAJobFeedback [-AppToken <String>] [-ComputerName <String>] -JobId <Int64> [-UseDefaultCredentials] [<CommonParameters>]
    
    
DESCRIPTION
    Returns feedback requested by jobs.
    

PARAMETERS
    -AppToken <String>
        An app token to access the UA API.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -ComputerName <String>
        The HTTP address of the UA REST API server.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Job <Job>
        The job to return feedback for.
        
        Required?                    true
        Position?                    named
        Default value                None
        Accept pipeline input?       True (ByValue)
        Accept wildcard characters?  false
        
    -JobId <Int64>
        The job ID to return feedback for.
        
        Required?                    true
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -UseDefaultCredentials [<SwitchParameter>]
        Use default credentials when connecting to the management API
        
        Required?                    false
        Position?                    named
        Default value                False
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer, PipelineVariable, and OutVariable. For more information, see
        about_CommonParameters (https://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    UniversalAutomation.Job
        
    
    
OUTPUTS
    System.Object
        
    
    
NOTES
    
    
        
    
    -------------------------- Example 1 --------------------------
    
    PS C:\> Get-UAJobFeedback -JobId 10
    
    Returns the feedback requested by job 10.
