# Get-UAStats

## SYNOPSIS
Returns stats about the UA instance.

## SYNTAX

```
Get-UAStats [-ComputerName <String>] [-AppToken <String>] [<CommonParameters>]
```

## DESCRIPTION
Returns stats about the UA instance.

## EXAMPLES

### Example 1
```
PS C:\> Get-UAStats
```

Returns stats about the UA instance.
