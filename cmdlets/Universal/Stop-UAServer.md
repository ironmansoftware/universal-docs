# Stop-UAServer

## SYNOPSIS
Stops the UA server.

## SYNTAX

```
Stop-UAServer [[-Server] <Process>] [-Force] [-ComputerName <String>] [-AppToken <String>] [<CommonParameters>]
```

## DESCRIPTION
Stops the UA server.

## EXAMPLES

### Example 1
```
PS C:\> Stop-UAServer
```

Stops the UA server.
