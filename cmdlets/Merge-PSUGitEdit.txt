
NAME
    Merge-PSUGitEdit
    
SYNOPSIS
    Commits and merges a Git edit into the repository.
    
    
SYNTAX
    Merge-PSUGitEdit [-AppToken <String>] [-ComputerName <String>] [-Credential <PSCredential>] [-HttpProtocolVersion <Version>] [-Integrated] -Message <String> [-Paths <String[]>] [<CommonParameters>]
    
    
DESCRIPTION
    Commits and merges a Git edit into the repository. This cmdlet will also push to the remote repository.
    

PARAMETERS
    -AppToken <String>
        An application token to use for authentication.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -ComputerName <String>
        The URL of the server to connect to.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Credential <PSCredential>
        The credential to use when connecting to the server.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -HttpProtocolVersion <Version>
        The HTTP protocol version to use when connecting to the server.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Integrated [<SwitchParameter>]
        For use when running within PowerShell Universal.
        
        Required?                    false
        Position?                    named
        Default value                False
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Message <String>
        The commit message to use when committing the changes.
        
        Required?                    true
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Paths <String[]>
        The paths to commit.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    <CommonParameters>

        OutBuffer, PipelineVariable, and OutVariable. For more information, see
        about_CommonParameters (https://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    None
        
    
    
OUTPUTS
    System.Object
        
    
    
NOTES
    
    
        
    
    -------------------------- Example 1 --------------------------
    
    PS C:\> Merge-PSUGitEdit -Message 'My commit message' -Paths 'C:\git\myrepo\file.txt'
    
    Commits and merges the changes in `file.txt` with the commit message `My commit message`.
