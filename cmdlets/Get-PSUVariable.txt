
NAME
    Get-UAVariable
    
SYNOPSIS
    Returns variables defined in UA.
    
    
SYNTAX
    Get-UAVariable [-Id] <Int64> [-AppToken <String>] [-ComputerName <String>] [-ValueOnly] [-UseDefaultCredentials] [<CommonParameters>]
    
    Get-UAVariable [-Name] <String> [-AppToken <String>] [-ComputerName <String>] [-ValueOnly] [-UseDefaultCredentials] [<CommonParameters>]
    
    
DESCRIPTION
    Returns variables defined in UA.
    

PARAMETERS
    -AppToken <String>
        An app token to access the UA API.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -ComputerName <String>
        The HTTP address of the UA REST API server.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Id <Int64>
        The ID of th variable.
        
        Required?                    true
        Position?                    0
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Name <String>
        The name of the variable.
        
        Required?                    true
        Position?                    0
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -ValueOnly [<SwitchParameter>]
        {{ Fill ValueOnly Description }}
        
        Required?                    false
        Position?                    named
        Default value                False
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -UseDefaultCredentials [<SwitchParameter>]
        Use default credentials when connecting to the management API
        
        Required?                    false
        Position?                    named
        Default value                False
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer, PipelineVariable, and OutVariable. For more information, see
        about_CommonParameters (https://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    None
        
    
    
OUTPUTS
    System.Object
        
    
    
NOTES
    
    
        
    
    -------------------------- Example 1 --------------------------
    
    PS C:\> Get-UAVariable
    
    Returns all variables.
    -------------------------- Example 2 --------------------------
    
    PS C:\> Get-UAVariable -Name 'username'
    
    Returns the username variable.
