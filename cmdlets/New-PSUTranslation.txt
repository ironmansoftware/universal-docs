
NAME
    New-PSUTranslation
    
SYNOPSIS
    Translation strings for PSU.
    
    
SYNTAX
    New-PSUTranslation [-AppToken <String>] [-ComputerName <String>] [-Integrated] -LanguageId <String> [-Name <String>] [-Strings <Hashtable>] [-UseDefaultCredentials] [<CommonParameters>]
    
    
DESCRIPTION
    Translation strings for PSU.
    

PARAMETERS
    -AppToken <String>
        An app token used to connect to the PSU server.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -ComputerName <String>
        The URL of the PSU server.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Integrated [<SwitchParameter>]
        No supported.
        
        Required?                    false
        Position?                    named
        Default value                False
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -LanguageId <String>
        The language ID used to look up the string dictionary.
        
        Required?                    true
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Name <String>
        A friendly name for the translation.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Strings <Hashtable>
        A hashtable of strings to available in this translation.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -UseDefaultCredentials [<SwitchParameter>]
        Whether to use default Windows credentials when connecting to PSU.
        
        Required?                    false
        Position?                    named
        Default value                False
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer, PipelineVariable, and OutVariable. For more information, see
        about_CommonParameters (https://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    None
        
    
    
OUTPUTS
    System.Object
        
    
    
NOTES
    
    
        
    
    -------------------------- Example 1 --------------------------
    
    New-PSUTranslation -LanguageId 'en-US' -Strings @{
        Title = "Title"
    }
    
    An English translation dictionary.
