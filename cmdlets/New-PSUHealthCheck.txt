
NAME
    New-PSUHealthCheck
    
SYNOPSIS
    Creates a new health check.
    
    
SYNTAX
    New-PSUHealthCheck [-AppToken <String>] [-ComputerName <String>] [-Description <String>] [-DisableUnix] [-DisableWindows] [-Disabled] [-Integrated] -Name <String> [-Options <Hashtable>] [-ScriptBlock <ScriptBlock>] [-UseDefaultCredentials] [<CommonParameters>]
    
    
DESCRIPTION
    Creates a new health check.

NOTES
    
    
        
    
    -------------------------- Example 1 --------------------------
    
    PS C:\> New-PSUHealthCheck -Name 'Test Health Check' -ScriptBlock {
        $Exists = Test-Path C:\Temp
        New-PSUHealthCheckResult -Message "C:\Temp exists: $Exists"
    }
    
    Creates a new health check named 'Test Health Check' that runs the Test-Path cmdlet.
