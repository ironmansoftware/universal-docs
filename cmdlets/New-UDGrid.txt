
NAME
    New-UDGrid
    
SYNOPSIS
    The grid is a 12-point grid system that can adapt based on the size of the screen that is showing the controls.
    
    
SYNTAX
    New-UDGrid [-Id <String>] [-ExtraSmallSize <Int32>] [-SmallSize <Int32>] [-MediumSize <Int32>] [-LargeSize <Int32>] [-ExtraLargeSize <Int32>] [-Item] [-Children <ScriptBlock>] [-ClassName <String>] [<CommonParameters>]
    
    New-UDGrid [-Id <String>] [-Container] [-Spacing <Int32>] [-Children <ScriptBlock>] [-ClassName <String>] [<CommonParameters>]
    
    
DESCRIPTION
    Creates a grid to layout components. The grid is a 12-point grid system that can adapt based on the size of the screen that is showing the controls.
