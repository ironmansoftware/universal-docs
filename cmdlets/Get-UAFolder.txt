
NAME
    Get-UAFolder
    
SYNOPSIS
    Returns folders defined within UA.
    
    
SYNTAX
    Get-UAFolder [-AppToken <String>] [-ComputerName <String>] [-Id <Int64>] [-UseDefaultCredentials] [<CommonParameters>]
    
    Get-UAFolder [-AppToken <String>] [-ComputerName <String>] [-Name <String>] [-UseDefaultCredentials] [<CommonParameters>]
    
    Get-UAFolder [-AppToken <String>] [-ComputerName <String>] [-Parent <Folder>] [-UseDefaultCredentials] [<CommonParameters>]
    
    
DESCRIPTION
    Returns folders defined within UA. Folders will be based on the internal and\or remote git repo.
    

PARAMETERS
    -AppToken <String>
        An app token to access the UA API.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -ComputerName <String>
        The HTTP address of the UA REST API server.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Id <Int64>
        The ID of the folder to return.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Name <String>
        The name of the folder to return.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Parent <Folder>
        The parent folder to return children for.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -UseDefaultCredentials [<SwitchParameter>]
        Use default credentials when connecting to the management API
        
        Required?                    false
        Position?                    named
        Default value                False
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer, PipelineVariable, and OutVariable. For more information, see
        about_CommonParameters (https://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    None
        
    
    
OUTPUTS
    System.Object
        
    
    
NOTES
    
    
        
    
    -------------------------- Example 1 --------------------------
    
    PS C:\> Get-UAFolder
    
    Returns all folders within UA.
    -------------------------- Example 2 --------------------------
    
    PS C:\> Get-UAFolder -Name 'Folder 1'
    
    Returns the folder with the name 'Folder 1'
    -------------------------- Example 3 --------------------------
    
    PS C:\> $Parent = Get-UAFolder -Name 'Parent'
    PS C:\> Get-UAFolder -Parent $Parent
    
    Returns all the children folders for the specified folder.
